            Notes for the Month

              Settings & GitHub Sync

              Fiscal Year

              Defaults to the current calendar year. Override here for this session only — resets on next load.

              GitHub Repository

              Data is stored as finance.json at the path you specify in your private repo. Token kept in your browser only. Create a fine-scoped PAT with Contents: Read and write permission on this repo only.

              Auto-saves to GitHub 90 seconds after your last change. Click Save to GitHub in the header to save now.

              Plaid (Phase 2)

              Plaid integration requires a small backend (Cloudflare Worker is recommended) so the Plaid client secret never lives in the browser. The Worker exposes three endpoints — /plaid/link-token, /plaid/exchange, /plaid/transactions — and the app calls them with your GitHub PAT as bearer auth. Until that's wired up, use Manual Import below or work with mock data.

              Manual Import

              CSV or OFX export from your bank. Useful when Plaid isn't set up yet, or for institutions Plaid doesn't cover well.

              Browser Lock

              Sets a password gate on this browser only. The real protection is the privacy of your repo — anyone with your PAT can read your data.

              Backup & Restore

              Plain JSON copy. Use before big changes; restore replaces all current data.

              Household Information

              Used in the Present view header.

              Advanced click to show — destructive operations

              Reset all data

              Replaces everything with mock starter data. Always back up first.

              About this dashboard

              Designed using Edward Tufte's principles of analytical design. Typeset in Warnock Pro, Interstate (with Condensed and Compressed variants), DIN Condensed Variable, and FF Meta Correspondence. Charts kept minimal: no gridlines, no chartjunk, axis-bound only. Data persists locally in your browser and (optionally) commits to your private GitHub repo as a single finance.json file.

              Build 2026-05-04-A · matches CMA Billing visual language · ships with mock data for prototype review.
